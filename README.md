# - `图书借阅管理系统`项目

## 项目简介

一个图书解约管理系统，作为初学JavaSpring的练手项目。其中包括了以下基础功能：
用户：登录与注册、个人中心、图书借阅、用户借阅信息
管理员：图书分类管理、图书借阅管理、图书存储管理、用户管理

## 目录结构说明

- documents -- 项目开发相关文档（如：项目需求文档、设计文档等等）
- projects -- 项目源码（比如：前端项目、后端项目等等）
